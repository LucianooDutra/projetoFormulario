# Projeto Formulário!


## Sobre o projeto:

  Projeto de um formulário, feito em JavaScript, que tem como principal objetivo enviar um formulário de avaliação da escola de magia "TrybeWarts"!


# Técnologias utilizadas:

 - JavaScript;
 - Linter;
 - CSS;
 - HTML;

# Habilidades trabalhadas:

 - Utilizar JavaScript;
 - Utilizar CSS;
 - Utilizar HTML;
